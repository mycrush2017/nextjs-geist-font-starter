BibleMee - Interactive Bible Study Website
=======================================

This is a single-page website that provides an interactive Bible study experience with multilingual support.

How to Use:
-----------
1. Extract all files from the ZIP archive
2. Open 'index.html' in a modern web browser (Chrome, Firefox, Safari, or Edge recommended)
3. The website should load immediately with all features available

Features:
---------
- Multilingual support (English, French, Spanish, and Haitian Creole)
- Interactive Bible study tools
- Audio content integration
- Responsive design that works on all devices
- Contact form
- Donation system
- Study resources and devotionals

Language Selection:
-----------------
- Use the language dropdown in the top navigation bar to switch between available languages
- Your language preference will be saved for future visits

Browser Support:
--------------
This website works best in modern browsers with JavaScript enabled. Recommended browsers:
- Google Chrome (latest version)
- Mozilla Firefox (latest version)
- Microsoft Edge (latest version)
- Safari (latest version)

Note:
-----
- The audio content links are placeholders and will need to be replaced with actual audio files
- The donation system is a demonstration and would need to be connected to a real payment processor
- The contact form is set up for demonstration purposes

For any questions or support, please use the contact form on the website.
